┌──────────────────────────────────────────────────────────────────────────────┐
│ 🚀 DESPLIEGUE RÁPIDO (5 minutos)                                            │
└──────────────────────────────────────────────────────────────────────────────┘

1. Generar certificados:
   bash mlops-stack/generate-certs.sh localhost

2. Validar configuración:
   docker-compose config > /dev/null

3. Construir e iniciar:
   cd mlops-stack && docker compose up -d

4. Configurar Keycloak (esperar 60s):
   bash mlops-stack/setup-keycloak.sh http://localhost:8080 admin admin123

5. Actualizar Client Secret en docker-compose.yml y reiniciar

6. Validar:
   curl -k https://localhost/mlflow/ | grep -i keycloak
